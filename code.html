								function fizzBuzz(fizzValue, buzzValue) {
									//Initialize array
									let returnArray = [];

									//Loop from 1 to 100
									for (let i = 1; i <= 100; i++) {

										//check to see if divisible by both 3 and 5
										//check t see if divisible by fizz value 3

										if (i % fizzValue == 0 && i % buzzValue == 0) {

											returnArray.push("Fizz");

											returnArray.push("Buzz");
										} else {
											returnArray.push(i);

FIZZ BUZZ

Fizz Buzz is an app that displays each number from 1 to 100.For each multiple of 3, display "Fizz" instead of the number. For each multiple of 5, display "Buzz" instead of the number. For numbers which are multiples of both 3 and 5, display "FizzBuzz" instead of the number.

Function: getValues()

  1. This function gets the fizz and buzz values from the user.
  2. The values are then parsed into an Integer, then validated to ensure they are numeric by using the Numeric property of the isInteger method.
  3. If the values are numeric, the fizzBuzz() function is called with two parameters (fizzValue and buzzValue) which are passed to the fizzBuzz() function. These parameters contain the input values from the user
  4. Then, displayData() function is called. This function will display the output on the screen.
  5. If not, an alert is generated with the message, "You must enter an integer.

Function: fizzBuzz()

  1. This function is initiated with two arguments, fizzValue and buzzValue.
  2. An empty array is initiated called returnArray.
  3. A for loop is used to loop throught the number 1 to 100.
  4. Each time through teh loop, an if statement checks to see if both fizzValue and buzzValue are divisible by 3 and 5 or by the numbers entered by the user. If that check evaluaes to a remainder of zero(0), the word "FizzBuzz" is pushed into the array called returnArray.
  5. An else if statement checks to see if fizzValue is divisible by 3. If that check evaluaes to a remainder of zero(0), the word "Fizz" is pushed into the array called returnArray.
  6. A second else if statement checks to see if buzzValue is divisible by 5. If that check evaluaes to a remainder of zero(0), the word "Buzz" is pushed into the array called returnArray.
  7. Lastly, if none of those conditions checks out, an else statement is used to pushed everything else into the array called returnArray.
  8. Finally, the array is returned and passed to the displayData() function with the parameter, fbArray.

Function: displayData()

  1. The function is implemented with an argument. The argument takes fbArray which was returned via the fizzBuzz() function.
  2. The tableBody and templateRow variables are initiated by retrieving the respective "results" and "fbTemplate" elements from the page
  3. The table is first cleared by setting it to blank.
  4. A for loop is used to loop through the fbArray and increment it by 5.
  5. The content in the template is retrieved via the document.importNode() medthod and assigned to tableRow.
  6. The table data (TDs) are extrated and assigned to an array called rowCols.
  7. The for loop cycles through the fbArray and assigns 5 records at a time to rowCols array.
  8. Likewise, the classList() method is used to add any CSS class to the TD tag each time through the array.
  9. Finally, the tableBody.appendChild() method is used to output the tableRow data to the screen.
